NEON CLOCK DASHBOARD

Neon Clock Dashboard is a modern, futuristic time utility web application built using React and Tailwind CSS. It combines multiple time-related features into a single visually appealing dashboard with a neon-glow, glassmorphism-inspired user interface. The goal of the project is to provide both functionality and aesthetics by integrating essential tools like a digital clock, analog clock, alarm system, stopwatch, and timer in one place.

FEATURES

The application includes a digital clock that displays the current local time along with UTC time in AM/PM format. It also features an digital clock . A fully functional stopwatch is included with start, pause, and reset capabilities. The timer allows users to set countdowns and alerts them when time is up.

The alarm system is one of the key features. Users can add multiple alarms, and each alarm will trigger a sound at the set time. There is also a stop feature to turn off the alarm when it rings. The system is designed to be simple and extendable, allowing future enhancements like snooze functionality.

Additionally, the dashboard supports Nepali date (Bikram Sambat), making it useful for users in Nepal. The interface is responsive and organized using a grid layout, ensuring it works well across different screen sizes.

TECH STACK

The project is built using React with Vite for fast development and performance. Tailwind CSS is used for styling, enabling quick design of modern UI elements such as gradients, neon glows, and glass effects. JavaScript (ES6+) is used for logic and functionality.

INSTALLATION AND SETUP

To run this project locally, first clone the repository to your system. Then navigate into the project folder and install all dependencies using npm install. After installation, start the development server using npm run dev. Once the server starts, open your browser and go to http://localhost:5173 to view the application.


ALARM FUNCTIONALITY

The alarm feature supports multiple alarms and plays a sound when triggered. Each alarm includes a stop option so users can easily turn it off. The system is designed in a way that additional features like snooze or custom alarm tones can be added later without major changes.
